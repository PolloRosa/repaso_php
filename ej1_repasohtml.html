Requerimiento funcional:

El profesor deja de tarea al estudiante crear 1 página HTML para repasar las etiquetas HTML5 básicas de encabezados, párrafo, listas y tabla.

Funcionalidad final:

Se creó 1 página HTML que contiene la estructura básica HTML5, los 6 encabezados h1 hasta h6, varios párrafos con información sobre atajos del IDE Visual Studio Code y etiquetas HTML, 1 lista ordenada, 1 lista desordenada, y 1 tabla que lista las etiquetas HTML5 para la creación de tablas con estilo provisto por las etiquetas <colgroup> y <col>.

Atajos importantes en Visual Studio Code

Ctrl+} para comentario en HTML.

Shift+1 para cargar toda la estructura básica de un archivo HTML

Shift+Alt+down o Shift+Alt+up para duplicar una línea

Ctrl+Alt+down o Ctrl+Alt+up para duplicar el cursor y escribir en varias líneas a la vez.

Alt+click para colocar el cursor en 2 lugares

Tablas

Elementos de una tabla.
TAG Significado Función
Table Table Tabla
TR Table row Fila
TH Table header Cabecera
TD Table data Celda
Caption Caption El subtítulo de una tabla. Se muestra centrado.
thead Table header Para agrupar el contenido de cabecera de una tabla.
tbody Table body Para agrupar el contenido del cuerpo de una tabla.
tfoot Table footer Para agrupar el contenido de pie de una tabla.
caption Caption Subtítulo o leyenda de una tabla. Se coloca inmediatamente después de <table>.
colgroup Group of columns Para especificar un grupo de 1 a más columnas en una tabla para darle estilo.
col Column Para especificar las propiedades de 1 o más columnas. Se usa dentro de <colgroup>.
