Diferencias en el uso de la pseudo-clase :first-child

Ejercicio 6 - Etiquetas HTML semánticas

Requerimiento funcional:

Hemos estudiado en clase las etiquetas HTML semánticas. El profesor dejó de tarea explicar cualquier tema en una página web usando etiquetas semánticas.

Etiquetas HTML en uso en el ejercicio:

  • <header>
  • <main>
  • <article>
  • <section>
  • <footer>

Funcionalidad final:

Se creó 1 página HTML que hace uso de las etiquetas listadas.

  • <header> y <footer> para los enlaces de inicio y fin de la página web.
  • <main> que contiene todo el contenido principal de la página web.
  • <article> que contiene el planteamiento del ejercicio y el ejercicio desarrollado.
  • <section> para las 2 secciones del plantemiento del ejercicio y del ejercicio desarrollado.

Definición

La pseudo-clase :first-child en CSS representa al primer elemento de entre un grupo de elementos hermanos. A continuación aprenderemos a usarlo de 2 maneras que suelen generar confusión.

div :first-child

Para seleccionar cualquier elemento anidado de un elemento div padre, que es el primer hijo de su padre.

div:first-child

Para seleccionar cualquier elemento div anidado de un elemento padre, que es el primer hijo de su padre.

Ejemplo

En la presente página web se han aplicado estilos para ambos selectores:

div.card :first-child {
color: darkred;
}

El script selecciona cualquier primer hijo dentro de todos los elementos div con clase .card y cambia su color de texto a rojo oscuro.

div.card:first-child {
background-color: rgba(120, 160, 236, 0.7);
}

El script selecciona el elemento div con clase .card que es primer hijo y cambia su color de fondo a celeste.
